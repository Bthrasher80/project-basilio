## Purpose

This document maps Linux Foundation Certified System Administrator (LFCS) objectives
to hands-on lab exercises performed on a dedicated Linux learning platform.
The goal is to demonstrate structured skill acquisition and operational relevance.

---

## Essential Commands (20%)

**Lab Activities**
- Filesystem navigation and permission management
- File search and text processing using grep, awk, and sed
- Archive creation and extraction
- Log review using journalctl and tail

**Outcome**
Demonstrates confidence with core Linux command-line workflows used in troubleshooting
and incident response.

---

## Operation of Running Systems (20%)

**Lab Activities**
- Systemd service creation and troubleshooting
- Process management and monitoring
- Scheduling with cron and systemd timers
- Resource monitoring under simulated load

**Outcome**
Builds operational awareness and system recovery skills.

---

## User and Group Management (10%)

**Lab Activities**
- Role-based user creation
- Sudo policy configuration
- Password aging and access auditing

**Outcome**
Reinforces access control and least-privilege principles.

---

## Networking (12%)

**Lab Activities**
- Static and dynamic IP configuration
- DNS troubleshooting
- Firewall rule creation
- Network diagnostics using ss, ip, and tcpdump

**Outcome**
Strengthens Linux networking fundamentals relevant to SOC investigations.

---

## Service Configuration (20%)

**Lab Activities**
- SSH hardening
- Web service deployment
- Local database service setup
- Container fundamentals using lightweight runtimes

**Outcome**
Demonstrates service administration and secure configuration practices.

---

## Storage Management (18%)

**Lab Activities**
- Partitioning and filesystem creation
- LVM volume management
- Filesystem resizing and mounts
- Disk pressure simulation

**Outcome**
Directly mirrors real-world incidents involving storage exhaustion and recovery.

---

## Summary

This approach ensures LFCS preparation is hands-on, repeatable, and directly applicable
to operational cybersecurity environments.
